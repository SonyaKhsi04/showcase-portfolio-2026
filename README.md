## Portfolio – Sonya Khosravani

Dit is mijn persoonlijke portfolio website, ontwikkeld als showcase van mijn werk en vaardigheden als UX/UI Designer en Front-end Developer.

## Over dit project

Deze portfolio is ontworpen om mijn creatieve en technische skills samen te brengen in één interactieve en visueel sterke website. Het project laat niet alleen mijn ontwerpen zien, maar ook mijn kennis van front-end development en interactieontwerp.

## Doel

Het belangrijkste doel van deze portfolio is om mij te helpen bij het vinden van een stageplek.
Ik wil hiermee laten zien wie ik ben, wat ik kan en hoe ik denk als ontwerper en developer.

Daarnaast helpt dit project mij om:

1. Mijn design skills verder te ontwikkelen
2. Mijn front-end development skills te verbreden
3. Beter te worden in het bouwen van interactieve webervaringen
4. Een professioneel en persoonlijk merk op te bouwen

## Gebruikte technologieën

- HTML5
- CSS3
- JavaScript (vanilla)
- Custom fonts en styling
- Intersection Observer API

## Features

- Interactieve homepage met animaties
- Carousel met projecten
- Projectpagina’s die dynamisch openen
- Scroll-animaties (fade-in effecten)
- Responsive design (in ontwikkeling/uitbreiding)
